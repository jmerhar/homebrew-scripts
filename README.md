## Installation

First, add the tap:

```bash
brew tap jmerhar/scripts
```

Then install any script by name:

```bash
brew install jmerhar/scripts/unlock-pdf
```

For more details on each script, see the [main repository](https://github.com/jmerhar/scripts).
